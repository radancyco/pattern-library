This pattern will allow you to place an accessible "Jump Menu" on your page that can reveal in-page content. Please read Accessibility & UX before using.

Notes

  1. Multiple Jump Menus per-page are allowed. Be sure to update aria-label with a unique name.
  2. Custom URL fragments are allowed. However, removing fragments is not permitted, as we want to be able to link to this content as well as allow users to share it.
  3. <div class="sr-only" aria-live="polite"></div> must always be included and be inclusively hidden. This div let's assistive technology users know there is new content.

Parameters

None. See Dependencies.
