Parameters

  • data-in-page-dynamic (Boolean, Mandatory): To be added to the parent element of the dynamic Jump Menu. In this case, the element with a class of .in-page.
  • data-in-page-name (String, Mandatory): To be added to each content container of the dynamic Jump Menu. In this case, the element with a class of .in-page__content. May contain any value you wish to see populate the select element.

Notes

  • Multiple Jump Menus per-page are allowed. However, be sure to update each aria-label with a unique name.
  • Custom URL fragments are allowed. Removing fragments is not permitted, as we want to be able to link to this content as well as allow users to share it. The dynamic version of the menu will produce generic fragments (e.g., content-1, content-2, etc.).
  • <div class="sr-only" aria-live="polite"></div> must always be included and be inclusively hidden. This div let's assistive technology users know there is new content.
