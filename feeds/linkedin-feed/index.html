This application may be used to generate a component that can be used to display a client's LinkedIn feed. The feed is generated via JavaScript and can be placed on any web page.

Dependencies

jQuery

jQuery is a dependency for this feed to run successfully, so be sure to include it on your page. TalentBrew has jQuery built in, so this step is not necessary on that platform. The script is deferred so that it may be placed anywhere on the page, but if the defer attribute is removed, then be sure to place the script after the jQuery library has been initialized.

Parameters

The following is a list of parameters that can be passed to the component to exhibit more control over the feeds output.

userid

String: Passing in the LinkedIn account id will display feed content from that account. This parameter is mandatory.

<script defer src="https://services.tmpwebeng.com/tmp-share/widget/js/linkedin-feed.js?userid=1843"></script>

Return to top

default-css

String: When set to y, a link call to ready-made CSS will be appended to the <head> element of the page the script is running on. This is a bare-bones layout for quick implementation. Developers may wish to use their own CSS or copy over these default styles and alter as needed.

<script defer src="https://services.tmpwebeng.com/tmp-share/widget/js/linkedin-feed.js?user=tmpww&default-css=y"></script>
