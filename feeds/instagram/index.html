This application may be used to generate a component that can be used to display a client's Instagram feed. The feed is generated via JavaScript and can be placed on any web page.

Table of Contents

  • Implementation
  • Dependencies
  • Parameters
    • user - Define which client feed to display
    • caption - Choose whether to "display" captions in feed
    • img - Choose which resolution of image to use
    • count - Choose amount of items to display in feed
    • default-css - Choose whether to load default CSS
  • Feedback

Dependencies

jQuery

jQuery is a dependency for this feed to run successfully, so be sure to include it on your page. TalentBrew has jQuery built in, so this step is not necessary on that platform. The script is deferred so that it may be placed anywhere on the page, but if the defer attribute is removed, then be sure to place the script after the jQuery library has been initialized.

Parameters

The following is a list of parameters that can be passed to the component to exhibit more control over the feeds output.

user

String: Passing in the Instagram account name will display feed content from that account. This parameter is mandatory.

<script defer src="https://services.tmpwebeng.com/tmp-share/widget/js/instagram.js?user=carrier"></script>

Return to top

caption

String: When set to n, the captions associated with each feed image will have special classes append to them. See below for more details. The default is y.

<script defer src="https://services.tmpwebeng.com/tmp-share/widget/js/instagram.js?user=carrier&caption=n"></script>

Accessibility

Please note that when the caption parameter is set to n, that the captions will still appear in each feed item, but have a class of "visually-hidden" and "wai" appended to them. It will then be up to the front-end developer to accessibly hide these captions from visual users only, as they are still required to aid assistive technology users. Do not use display: none to hide these feed captions.

Return to top

img

String: Determines which resolution of the image to be used. Possible values include low, high, and thumb. The default is thumb, which will render thumbnail images.

<script defer src="https://services.tmpwebeng.com/tmp-share/widget/js/instagram.js?user=carrier&img=low"></script>

Return to top

count

Number: Sets the number of items to be displayed in the feed. The default is 10.

<script defer src="https://services.tmpwebeng.com/tmp-share/widget/js/instagram.js?user=carrier&count=5"></script>

Return to top

default-css

String: When set to y, a link call to ready-made CSS will be appended to the <head> element of the page the script is running on. This is a bare-bones layout for quick implementation. Developers may wish to use their own CSS or copy over these default styles and alter as needed.

<script defer src="https://services.tmpwebeng.com/tmp-share/widget/js/instagram.js?user=carrier&default-css=y"></script>
