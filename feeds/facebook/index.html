--- jquery: true title: "Facebook Feed 1.0" ---

This application may be used to generate a component that can be used to display a client's Facebook feed. The feed is generated via JavaScript and can be placed on any web page.

Table of Contents

  • Implementation
  • Dependencies
  • Parameters
    • user - Define which client feed to display
    • show-url - Choose whether to "display" URL under image in feed
    • img - Choose which resolution of image to use
    • count - Choose amount of items to display in feed
    • default-css - Choose whether to load default CSS
  • Feedback

Dependencies

jQuery

jQuery is a dependency for this feed to run successfully, so be sure to include it on your page. TalentBrew has jQuery built in, so this step is not necessary on that platform. The script is deferred so that it may be placed anywhere on the page, but if the defer attribute is removed, then be sure to place the script after the jQuery library has been initialized.

Parameters

The following is a list of parameters that can be passed to the component to exhibit more control over the feeds output.

user

String: Passing in the Facebook account name will display feed content from that account. This parameter is mandatory.

<script defer src="https://services.tmpwebeng.com/tmp-share/widget/js/facebook.js?user=radancyco"></script>

Return to top

show-url

String: When set to n, the Facebook url associated with each feed image will have special classes append to them. See below for more details. The default is y.

<script defer src="https://services.tmpwebeng.com/tmp-share/widget/js/facebook.js?user=radancyco&show-url=n"></script>

Accessibility

Please note that when the show-url parameter is set to n, that the Facebbok URL will still appear below each image within the feed, but have a class of "visually-hidden" and "wai" appended to it. It will then be up to the front-end developer to accessibly hide this text from visual users only, as the URL is still required to aid assistive technology users. Do not use display: none to hide these facebook URL.

Return to top

img

String: Determines which resolution of the image to be used. Possible values include low, high, and thumb. The default is thumb, which will render thumbnail images.

<script defer src="https://services.tmpwebeng.com/tmp-share/widget/js/facebook.js?user=radancyco&img=low"></script>

Return to top

count

Number: Sets the number of items to be displayed in the feed. The default is 10.

<script defer src="https://services.tmpwebeng.com/tmp-share/widget/js/facebook.js?user=radancyco&count=5"></script>

Return to top

default-css

String: When set to y, a link call to ready-made CSS will be appended to the <head> element of the page the script is running on. This is a bare-bones layout for quick implementation. Developers may wish to use their own CSS or copy over these default styles and alter as needed.

<script defer src="https://services.tmpwebeng.com/tmp-share/widget/js/facebook.js?user=radancyco&default-css=y"></script>
