This feed may be used to generate a list of tweets (a 140 character or less message) from an individual Twitter account. The feed can then be linked to directly or referenced within an inline frame (iframe).

Table of Contents

  • Initial Setup
  • Parameters & Examples
    • clear - Clear cleint's feed cache
    • client - Define which client feed to view
    • css - Inject custom CSS (Cascading Style Sheets) into the feed
    • ga - Apply Google Analytics tracking code
    • img - Allow images to load in feed
    • tab - Disable links from opening in new tab/window
  • Feedback

Initial Setup

While the feed can be viewed directly, the most common use case is to load this content within an iframe element by supplying an absolute path to the feed application. A title attribute and value must also be provided.

<iframe loading="lazy" src="https://services1.tmpwebeng.com/widget/twitter" title="[Company Name] Twitter Feed"></iframe>

However, the feed will not render usable content until we begin passing parameters to it.

Accessibility

Please note use of the title attribute in all code examples. A brief description of the contents of the iframe element must be provided to assist those who are using assistive technology, such as screen readers, to access the content.

Parameters & Examples

The following is a list of parameters that can be passed to the feed to render content and to allow for better control over its presentation and behavior. All functions must be used with the client parameter. Code and output examples provided where applicable.

clear

Boolean: When set to true, the server cache for selected client is cleared. The default is false.

<iframe loading="lazy" src="https://services1.tmpwebeng.com/widget/twitter?client=radancyco&ga=UA-000000-01" title="{{ site.company-name }}'s Twitter Feed"></iframe>

Return to top

client

String: Accepts any valid Twitter username value.
