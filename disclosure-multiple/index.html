This pattern will allow you to easily place and control the presentation of a multiple dislosure user interface (UI). Please read Accessibility & UX before using.

Parameters

  • data-disclosure-multiple-icon (Boolean, Optional): Permits inclusion of a visual icon that changes when the control is clicked. Any icon can be included, via the CSS.
  • data-disclosure-multiple-open (Number, Optional): Specifies which tab should be opened on page load.

Note: The "Toggle All" button is optional.
