Properties

Use the following properties to configure this component.

Elements
Element Note Description
video Optional, Hardcoded Include this element when a background video is needed. Must include .animation-toggle__video to be used by the script.
track Optional, Hardcoded Include this element as a child of any video that requires audio descriptions. Must point to a valid VTT file.
Attributes
Attribute Apply Note Description
aria-label video String, Optional, Hardcoded Used to pass custom label to video element. This will override the default label, which is "Background Animation".
data-audio-description-button .animation-toggle String, Optional, Hardcoded Used to pass custom label to .animation-toggle__audio. This will override the default label, which is "Audio Description".
data-media video String, Optional, Hardcoded Used to indicate at which point data-srcset will load. Accepts any valid media query, but typically min-width will be used.
data-pause-button .animation-toggle String, Optional, Hardcoded Used to pass custom label to .animation-toggle__pause. This will override the default label, which is "Pause Animation".
data-poster video Number, Optional, Hardcoded Used to indicate at which point video will load when paused. Accepts any valid time increment. Video will start from this point when play is enabled. To test, pause video and refresh page.
data-srcset video String, Optional, Hardcoded Used to reference video asset that will load when data-media is present. Typically, this will reference the larger video, while the src attribute will reference the smaller video.
Classes
Class Apply Note Description
.animation-toggle div, section Mandatory, Hardcoded Used on the parent element that wraps your content and animation(s).
.animation-toggle__audio button Optional, Dynamic Used on all audio description buttons.
.animation-toggle__controls div Mandatory, Dynamic Used to contain pause and audio description buttons.
.animation-toggle__description div Optional, Hardcoded Used on all div elements that are children of .animation-toggle. The aria-live attribute is required on this element when used.
.animation-toggle__backdrop .animation-toggle Optional, Hardcoded Append to .animation-toggle class to enable. This will enable video to animate in on page load and for background shading when video is paused. Often used on hero banner.
.animation-toggle__pause button Mandatory, Dynamic Used on all pause buttons.
.animation-toggle__video video Mandatory, Hardcoded Used on all video elements that are children of .animation-toggle.
.animation-enabled body Mandatory, Dynamic Applied on page load. CSS animation will only run when this class is present. Removing it will pause animation. By default, we are putting accessibility first by treating animation as an enhancement.

Additional Notes

  • Video should always be delivered as an optimized webm or mp4. The file size should be as minimal as possible without sacrificing quality, to reduce overall page weight. Multiple source elements are not supported at this time.
  • Video should be no longer than 15-30 seconds in length. The shorter the video is, the less of an impact it will have on performance.
  • In addition to the pause button, video and CSS animations utilize prefers-reduced-motion. When a user disables animation in their operating system, the animation will not autoplay.
  • Pausing one video pauses all videos that use the script across the site and on return visits.
