Parameters

  • data-disclosure-icon (Boolean, Optional): Use to include a visual icon that changes when the control is clicked. Any icon can be included, via the CSS.

Classes

  • .disclosure--heading: Include this class on any heading you might have that wraps your button. For example, a FAQ may require interactive headings, like so: <h2 class="disclosure--heading"><button class="disclosure--btn"> ... </button></h2>
