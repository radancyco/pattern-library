Classes

  • disclosure: Include this class on the parent element that wraps multiple disclosures. If "Show All" button is removed, this class is not needed.
  • disclosure--btn: Include this class on your disclosure button.
  • disclosure--content: Include this class on your disclosure content.
  • disclosure--heading: Include this class on any heading you might have that wrap your button.

Parameters

  • data-disclosure-enable-url (Boolean, Optional): Used to include a URL fragment on click. Including this can be useful for sharing specific in-page content fragments.
  • data-disclosure-icon (Boolean, Optional): Used to include a visual icon that changes state when the disclosure is clicked. Any icon can be included via CSS.
  • data-disclosure-open (Boolean, Optional): Specifies which disclosures should be opened on page load.
  • data-disclosure-viewport (Boolean, Optional): Used with the disclosure--heading class, this will maintain button interactivity on mobile, while removing it on desktop.
