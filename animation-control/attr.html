Properties

Use the following properties to configure this component. Unless otherwise noted, all properties are mandatory.

Classes
Class Description
.ac Include this class on the parent element that wraps your content and animation.
.ac__button This class is included on all pause buttons generated by the script.
.ac__video Include this class on all video elements, when present.
.animation-enabled This class is applied to the body element on page load. CSS animation will only run when this class is present. Removing it will pause animation. By default, we are putting accessibility first by treating animation as an enhancement.
Elements
Element Note Description
video Optional Include this element if choosing to include animation via HTML video. The following attributes will be required: aria-hidden="true", crossorigin, disableremoteplayback, loop, muted, playsinline, and tabindex="-1". If you validate your page, disableremoteplayback may produce an error. However, this can be ignored.

Notes

  1. Video should be delivered as an optimized webm or mp4. The file size should be reduced as much as possible to reduce overall page weight.
  2. Video should be no longer than 20-30 seconds in length. The shorter the video is, the less of an impact it will have on performance.
  3. In addition to the pause button, video and CSS animations utilize prefers-reduced-motion. When a user disables animation in their operating system, the animation will not autoplay, nor will the pause button be present.
  4. A cookie called AnimationPaused is added or removed when pause button is toggled. This cookie remembers the desired setting on other pages of the site that use the same script and on return visits.
  5. Video may not autoplay if mobile device is in low-power mode. This is so that battery life is not unnecessarily darined.
