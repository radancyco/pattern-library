Accessibility & UX

  • Jump Menus that utilize a single select dropdown to link to external pages, must always include a submit button. Unexpectedly taking a user to another page, upon making a selection from an input control, will cause a change of context, which is a failure of Success Criterion 3.2.2: On Input. If you wish to navigate away from a page without a submit button, then use a hyperlink or a disclosure that reveals a list of hyperlinks to other pages, when opened.
  • Complex forms that change content based on multiple selections, must also contain a submit button. Changing content on the fly, based on each selection made, can be confusing for everybody.
  • The only exception to the above rules is in-page navigation. If you are using a single select dropdown to change content within the page, then a submit button is not required. Please see Jump Menu (In-page) for more details.
